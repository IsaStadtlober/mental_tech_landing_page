# 🧠 Mental Tech Landing Page

Landing page para a Mental Tech, com foco em acessibilidade, linguagem clara e uma experiência visual acolhedora para famílias e escolas.

## 🚀 Tecnologias

- **React** - Framework de desenvolvimento web
- **TypeScript** - Linguagem de programação orientada a objetos
- **Vite** - Ambiente de desenvolvimento front-end
- **Framer Motion** - Biblioteca de animações
- **Fontsource (tipografias locais)** - Biblioteca de fontes
- **Tailwind CSS** - Framework de estilização CSS

## ▶️ Como executar

**1. Instale as dependências:**
   npm install
**2. Inicie o projeto:**
   npm run dev
**3. Abra o endereço exibido no terminal.**

## 🛠️ Scripts

- **npm run dev:** inicia o ambiente local
- **npm run build:** gera a build de produção
